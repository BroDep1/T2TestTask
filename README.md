Для запуска сервиса введите `docker-compose up`\
Open-Api документация: http://localhost:8080/swagger-ui/index.html \

Сервис создан c использованием Spring MVC. Использование WebFlux в данном случае не целесообразно, так как значительно повышает порог входа на проект, а также сложность и читаемость кода. При этом с webflux сильного выигрыша в перфомансе не будет, так как в этом сервисе мною реализована ассинхронная обработка файлов, и spring mvc для этого достаточно.
